# 项目描述

> 这是一个用VUEJS写高仿网易云音乐的webapp，只实现了APP的核心功能


## 安装运行（Build Setup）

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```

## 功能

- [ ]  实现音乐播放器的播放模式调整
- [ ]  搜索模块
